Proyecto de Ejemplo
Este es un archivo de inicio para el proyecto de ejemplo. A continuación se describen los detalles básicos y la estructura del proyecto.

Objetivo del Proyecto: Este repositorio está diseñado para demostrar cómo crear y resolver conflictos en Git utilizando la consola de Git.

Estructura del Proyecto:

main: La rama principal que contiene la versión estable del proyecto.
rama-conflicto: Una rama secundaria creada para introducir y resolver conflictos como parte de un tutorial.
Pasos para Contribuir:

Clona el repositorio.
Crea una nueva rama para tus cambios.
Realiza los cambios y realiza confirmaciones frecuentemente.
Fusiona tus cambios en la rama principal, resolviendo cualquier conflicto que pueda surgir.
